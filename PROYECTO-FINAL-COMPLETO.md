## 🚀 CÓMO INICIAR TODO

### 1. Backend
```bash
cd backend
npm install  # Si no está instalado
npm run dev
```
**URL:** `http://localhost:3001`

### 2. Frontend Web
```bash
cd frontend-web
npm install  # Si no está instalado
npm start
```
**URL:** `http://localhost:3000`

### 3. Mobile Android
```bash
cd mobile-android
npm install  # Si no está instalado
npm start    # Metro
npx react-native run-android
```

### 4. Mobile iOS (macOS)
```bash
cd mobile-android
npm install  # Si no está instalado
cd ios && pod install && cd ..
npm start
npx react-native run-ios
```

### 5. Desktop
```bash
cd desktop-windows
npm install  # Si no está instalado
npm run dev
```
